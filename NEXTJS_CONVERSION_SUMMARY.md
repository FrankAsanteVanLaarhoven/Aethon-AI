## 🚀 **Getting Started**

### **Installation**
```bash
cd frontend
npm install
```

### **Environment Setup**
```bash
cp env.example .env.local
# Edit .env.local with your configuration
```

### **Development**
```bash
npm run dev
# Open http://localhost:3000
```

### **Production Build**
```bash
npm run build
npm run start
```
